Foi meu primeiro projeto da faculdade, desenvolvido na disciplina de Frontend e Frameworks. O objetivo era ajudar cadeirantes e deficientes físicos a encontrar rotas e locais acessíveis, usando a API Mapbox para traçar rotas e pontos de acessibilidade, além de React para o site e React Native para o app.

Facilitando a Locomoção com Tecnologia

O projeto "Mobilidade Acessível" surgiu inicialmente como um desafio de design, com o objetivo de criar interfaces para dispositivos móveis, computadores e smartwatches, utilizando a ferramenta Figma. Posteriormente, na disciplina de Frontend, a missão foi expandida para a implementação prática dessas telas, transformando-as em sites funcionais para computadores e dispositivos móveis.

Durante o desenvolvimento, além da necessidade de criar o frontend completo do site, busquei integrar funcionalidades essenciais para a experiência do usuário, com foco em acessibilidade. Um dos maiores desafios foi garantir que a funcionalidade de GPS funcionasse corretamente. Para isso, explorei várias APIs gratuitas e optei pela integração com o Mapbox, que ofereceu recursos como geolocalização, rotas e até autocomplete nas buscas, tornando o site ainda mais intuitivo e acessível.

O resultado final permite que o usuário compartilhe sua localização atual e trace rotas acessíveis para seu destino. A seguir, você encontrará o link para o site hospedado gratuitamente no Netlify, onde é possível explorar o projeto de forma interativa.
